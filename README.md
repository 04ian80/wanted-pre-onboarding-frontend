# 원티드 프리온보딩 프론트엔드 4월 과제

원티드 프리온보딩 프론트엔드 4월 과제 레파지토리입니다.

`git clone` 후 아래 순서대로 터미널에 명령어를 입력하시면 서버가 실행됩니다.

```
git clone https://github.com/04ian80/wanted-pre-onboarding-frontend.git
```
```
npm install
npm start
```
